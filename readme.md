## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** JavaScript
- **Package Manager:** Npm

### Installation

Build AWS-ESP32-Server from the source and install dependencies:

1. **Clone the repository:**

   ```sh
   git clone https://github.com/Jazzel/AWS-ESP32-Server
    ```

2. **Navigate to the project directory:**

    ```sh
    cd AWS-ESP32-Server
    ```
3. **Install the dependencies:**

    Using npm:
    ```
    npm install
    ```
4. **Start the server:**

    ```sh
    npm start
    ```
5. **Access the server:**
    Open your web browser and navigate to `http://localhost:3000` to access the server.

### Usage
To use the AWS-ESP32-Server, you can send POST requests to the server with JSON data. The server will log the received data and respond accordingly.
